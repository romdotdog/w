### what's this?

this a compiler for a language that

- compiles to and is idiomatic to WebAssembly
- has Rust-like syntax but C-like semantics
- has manual memory management
- explicit mutability
  - e.g. `let mut`, `*mut x`
- global namespacing
  - without headers!
